# 展锐 Spreadtrum 教程

## 概述

展锐 (Spreadtrum/UNISOC) 平台使用 BSL (Boot Strap Loader) + FDL (Flash Download) 协议进行刷机操作。

## 支持的芯片

| 系列 | 芯片型号 |
|------|----------|
| 虎贲 T7 | T760, T770 |
| 虎贲 T6 | T610, T618, T700, T710, T740 |
| SC | SC9863A, SC9832E, SC7731E |
| 其他 | SC7731, SC8541 |

## 进入下载模式

### 方法一：组合键

1. 完全关机
2. 按住 **音量-**
3. 保持按住，插入 USB 数据线
4. 等待电脑识别

### 方法二：工程模式

部分设备可通过拨号盘进入：
```
*#*#83781#*#*
```

## 验证连接

设备管理器中应显示：
- `SPRD U2S Diag` (COM端口)
- 或 `Spreadtrum Phone`

## 使用 SakuraEDL

### 1. 选择 Spreadtrum 平台

打开 SakuraEDL，点击 **Spreadtrum** 选项卡。

### 2. 连接设备

1. 点击 **连接** 按钮
2. 让设备进入下载模式
3. 软件自动检测并显示设备信息

### 3. 加载固件

#### PAC 文件（推荐）

1. 点击 **加载 PAC**
2. 选择 `.pac` 固件包
3. 分区列表自动显示

#### 单独镜像

1. 选择目标分区
2. 点击 **选择文件**
3. 选择对应镜像

### 4. 刷写操作

1. 勾选要刷写的分区
2. 点击 **开始下载**
3. 等待完成

## PAC 文件说明

PAC 是展锐的官方固件格式，包含：
- 分区表信息
- 所有分区镜像
- 刷机配置

### PAC 文件结构

```
firmware.pac
├── fdl1.bin      # 第一阶段加载器
├── fdl2.bin      # 第二阶段加载器
├── boot.img      # 内核
├── system.img    # 系统
├── vendor.img    # 厂商
├── userdata.img  # 用户数据
└── ...
```

## 常见操作

### 刷写完整固件

1. 加载 PAC 文件
2. 勾选所有分区
3. 点击下载

### 刷写单个分区

1. 选择目标分区
2. 选择镜像文件
3. 点击刷写

### 读取分区

1. 选择要备份的分区
2. 点击 **读取**
3. 选择保存位置

### 擦除数据

1. 选择 `userdata` 分区
2. 点击 **擦除**

## FDL 地址配置

不同芯片的 FDL 加载地址不同：

| 芯片 | FDL1 地址 | FDL2 地址 |
|------|-----------|-----------|
| T760 | 0x5500 | 0x9EFFFE00 |
| T618 | 0x5500 | 0x8E000000 |
| SC9863A | 0x5500 | 0x8E000000 |

::: tip 提示
SakuraEDL 会自动检测芯片并配置正确的地址，一般无需手动设置。
:::

## 常见问题

### Q: 设备无法识别？

1. 检查驱动是否安装
2. 尝试不同的 USB 端口
3. 检查是否正确进入下载模式

### Q: 下载失败？

1. 检查固件是否匹配设备
2. 尝试重新进入下载模式
3. 检查 USB 数据线

### Q: PAC 加载失败？

1. 确认 PAC 文件完整
2. 检查文件是否损坏
3. 尝试重新下载固件

## 注意事项

::: warning 警告
- 刷机前备份重要数据
- 使用正确的固件版本
- 不要中断刷机过程
- 确保电池电量充足
:::

::: danger 危险分区
- `fdl1` / `fdl2` - 加载程序
- `spl` - 二级引导
- `nvitem` - NV 数据
- `phasecheck` - 校准数据
:::
